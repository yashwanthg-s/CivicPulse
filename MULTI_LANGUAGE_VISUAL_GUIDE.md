## Data Flow

```
┌─────────────────────────────────────────────────────────┐
│                  LANGUAGE SERVICE                       │
│  ┌──────────────────────────────────────────────────┐  │
│  │ translations = {                                 │  │
│  │   en: { key1: 'English', key2: 'English', ... } │  │
│  │   kn: { key1: 'ಕನ್ನಡ', key2: 'ಕನ್ನಡ', ... }     │  │
│  │   hi: { key1: 'हिंदी', key2: 'हिंदी', ... }     │  │
│  │ }                                                │  │
│  └──────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│              LANGUAGE CONTEXT                           │
│  ┌──────────────────────────────────────────────────┐  │
│  │ language = 'en' | 'kn' | 'hi'                   │  │
│  │ t(key) → returns translation for current lang   │  │
│  │ setLanguage(lang) → changes language            │  │
│  └──────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│              COMPONENTS                                 │
│  ┌──────────────────────────────────────────────────┐  │
│  │ const { t } = useLanguage()                      │  │
│  │ <h1>{t('submitComplaint')}</h1>                 │  │
│  │ <button>{t('loginButton')}</button>             │  │
│  └──────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│              BROWSER DISPLAY                            │
│  ┌──────────────────────────────────────────────────┐  │
│  │ Displays text in selected language              │  │
│  │ English / ಕನ್ನಡ / हिंदी                          │  │
│  └──────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

## Storage

```
Browser localStorage
├── preferredLanguage: 'en' | 'kn' | 'hi'
├── user: { id, username, role, ... }
└── currentPage: 'complaint' | 'history' | ...
```

## Summary

The multi-language feature provides:
- ✓ Language selection during citizen login
- ✓ Automatic language persistence
- ✓ Beautiful modal UI
- ✓ Support for English, Kannada, and Hindi
- ✓ Easy to extend with more languages
- ✓ Smooth animations and transitions
