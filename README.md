## ⚡ Key Features

<br/>

<div align="center"><h3><code>01</code> — 🤖 AI Complaint Categorization</h3></div>

> CivicPulse uses **state-of-the-art NLP** (HuggingFace Transformers + spaCy) to read complaint text and instantly route it to the right department — zero human intervention.

```
┌──────────────────────────────────────────────────────────────────────┐
│  INPUT  →  "Garbage not collected near Bus Stand for 3 days"         │
├──────────────────────────────────────────────────────────────────────┤
│  NLP ENGINE PROCESSING...                                            │
│  ├── Entity Recognition     → Location: Bus Stand                    │
│  ├── Intent Classification  → Category: Waste Management            │
│  └── Urgency Detection      → Severity: HIGH (3 days overdue)       │
├──────────────────────────────────────────────────────────────────────┤
│  OUTPUT →  ✅ Dept: Sanitation | Priority: 🔴 HIGH | SLA: 24 hrs    │
└──────────────────────────────────────────────────────────────────────┘
```

---

<div align="center"><h3><code>02</code> — 📍 EXIF Location Intelligence</h3></div>

> Citizens just upload a **photo**. CivicPulse extracts GPS coordinates from the image's EXIF metadata — no manual entry needed.

```
📷  Photo Uploaded by Citizen
     │
     ├──  ✅  GPS Metadata Found   ──►  📌  Auto-pin on interactive map
     │
     └──  ❌  No GPS Metadata      ──►  🗺️  Manual location selector shown

  ⚠️  SMART FRAUD DETECTION:
     If uploaded image GPS ≠ manually entered location  ──►  🚩  Flagged for Admin Review
```

---

<div align="center"><h3><code>03</code> — ⚡ Smart Priority Scoring Engine</h3></div>

> A dynamic algorithm scores every complaint so authorities **always work on what matters most**.

<div align="center">

| Signal | Weight | Example |
|--------|--------|---------|
| 🔴 Issue Severity | 35% | Road cave-in vs. minor pothole |
| 📍 Complaint Clusters | 25% | 10 reports for same location |
| 🏥 Location Sensitivity | 20% | Near hospital, school, highway |
| ⏰ SLA Deadline Breach | 20% | 48+ hours without response |

```
Priority Score  =  (Severity × 0.35) + (Clusters × 0.25) + (Sensitivity × 0.20) + (SLA × 0.20)
```

</div>

---

## 🔐 Security

<div align="center">

| Layer | Mechanism | Protection |
|-------|-----------|------------|
| 🔑 Authentication | JWT with expiry + refresh | Prevents unauthorised access |
| 📁 File Uploads | Multer — type, size, extension validation | Blocks malicious file uploads |
| 📍 Location Trust | GPS ↔ manual cross-validation | Detects fraud, geo-spoofing |
| 🤖 AI Integrity | Human detection on every image | Stops spam & irrelevant submissions |
| 🔁 Duplicate Guard | Jaccard + Haversine threshold checks | Blocks repeat submissions |
| 🛡️ Input Sanitisation | Server-side validation before DB writes | Prevents XSS & injection |
| 🗄️ Database | Parameterised queries (MySQL) | SQL injection prevention |
| 🌐 CORS Policy | Strict origin whitelisting on Express | Blocks unauthorised API callers |

## 📡 API Snapshot

```
  BASE URL:  http://localhost:5000/api
  ────────────────────────────────────────────────────────────────────────────

  AUTH
  POST    /auth/register              →  Register new citizen account
  POST    /auth/login                 →  Login + receive JWT token

  COMPLAINTS
  POST    /complaints                 →  Submit complaint (multipart/form-data)
  GET     /complaints                 →  List all complaints (admin)
  GET     /complaints/:id             →  Get single complaint + full status
  PATCH   /complaints/:id/status      →  Update complaint status (admin)
  GET     /complaints/heatmap         →  Geo-coordinates for heatmap render
  GET     /complaints/priority-queue  →  Priority-sorted complaint list

  AI SERVICE  ·  FastAPI  ·  port 8000
  POST    /analyse                    →  NLP categorisation + priority score
  POST    /detect-human               →  Skin tone / human presence check
  POST    /detect-duplicate           →  Jaccard + Haversine duplicate check
  POST    /extract-exif               →  GPS metadata extraction from image

  ────────────────────────────────────────────────────────────────────────────
  🔒  All /complaints routes require:   Authorization: Bearer <JWT>
```
